# Support

Last reviewed: 2026-07-15

Status: **Maintained support reference**

## Product And Membership Support

For membership approval, moderation, safeguarding, local policy, billing, or
community data, use the Contact, Help Centre, Knowledge Base, Trust and Safety,
or Report a Problem route supplied by your Project NEXUS community. Repository
maintainers cannot make tenant governance decisions.

## Software Defects

Open an issue in the repository with:

- the edition/surface and source SHA if known;
- the page or API method/path;
- the community/tenant slug without personal data;
- steps to reproduce, expected result, and actual result;
- browser/OS or runtime versions; and
- request/correlation ID and sanitized logs where relevant.

Do not publish passwords, tokens, private messages, database rows, identity or
safeguarding evidence, provider secrets, production IP access details, or
another person's personal information.

For a suspected vulnerability, do not open a public issue; follow
[SECURITY.md](SECURITY.md).
